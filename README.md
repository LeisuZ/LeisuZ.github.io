# 江西“两山”理论实践可视化报告

## 项目简介

本项目是一个关于江西"两山"理论（绿水青山就是金山银山）实践的可视化报告，旨在通过数据可视化、交互式体验和深度案例展示，全面呈现江西在生态文明建设方面的成果和经验。

## 项目功能

### 核心功能

1. **头部区域**：展示项目标题、生态指数和习近平总书记的"两山"理念引用
2. **关键生态指标**：以卡片形式展示江西森林覆盖率、生态产品总值、年碳汇量等核心指标
3. **时空对比维度**：通过图像对比滑块展示江西生态环境变化，以及习近平生态文明思想与江西实践的时间轴
4. **数据可视化图表**：使用ECharts实现多种图表，包括森林覆盖率动态趋势、生态产品总值构成、地市森林覆盖率对比等
5. **江西实践亮点案例**：展示抚州"两山银行"、资溪县GEP核算、江西省"生态云"平台等创新实践
6. **生态价值转化典型模式**：介绍生态资源资产化、生态产业融合、生态补偿和数字化赋能四种模式
7. **生态数字孪生与智慧监测模块**：展示江西生态"一张图"和实时数据监测
8. **碳汇价值模拟计算器**：用户可以输入林地面积和保护年限，计算碳汇价值
9. **深度案例与故事线**：包括生态账本模拟计算器、"一湖清水"守护故事、鄱阳湖国家湿地公园介绍和江西环境变化视频展示
10. **生态文明建设时间轴**：展示江西生态文明建设的重要事件和里程碑
11. **数字化赋能展望**：探讨数字化技术在生态价值核算与转化中的应用
12. **绿水青山知识小测验**：通过互动测验帮助用户学习"两山"理念相关知识

## 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **图表库**：ECharts 5.4.3
- **图标库**：Font Awesome 6.4.0
- **响应式设计**：适配不同屏幕尺寸

## 安装与使用

### 本地运行

1. **克隆或下载项目**：将项目文件下载到本地
2. **启动本地服务器**：
   - 使用Python内置服务器：
     ```bash
     python -m http.server 8080
     ```
   - 或使用Node.js的http-server：
     ```bash
     npx http-server -p 8080
     ```
3. **访问项目**：在浏览器中打开 `http://localhost:8080`

### 直接打开

对于简单浏览，也可以直接在浏览器中打开 `index.html` 文件，但某些功能（如地图加载）可能会受到浏览器安全策略的限制。

## 功能模块说明

### 1. 数据可视化图表

项目使用ECharts实现了多种图表，包括：
- 森林覆盖率动态趋势（2010-2025）
- 生态产品总值（GEP）构成
- 部分地市森林覆盖率对比
- 林业碳汇交易增长趋势
- 生态旅游收入增长趋势
- 空气质量指数变化
- 鄱阳湖湿地生态变化
- 江西绿色产业产值增长
- 江西生态指标 vs 全国平均

### 2. 交互式功能

- **图像对比滑块**：通过拖动滑块，直观对比江西生态环境的变化
- **碳汇价值计算器**：输入林地面积和保护年限，计算碳汇价值
- **生态账本模拟器**：通过调整林地面积，模拟存入"两山银行"的预期收益
- **知识小测验**：回答关于"两山"理念的问题，获得即时反馈

### 3. 深度案例

- **"一湖清水"守护故事**：展示鄱阳湖生态保护和退捕渔民转产就业的成果
- **鄱阳湖国家湿地公园**：介绍公园的核心景点和生态价值
- **江西环境变化视频**：通过视频展示江西生态环境治理的成效
